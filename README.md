# TecNardi

Protótipo de site educativo sobre eletrônica, informática e arcade.

## Estrutura criada
- `index.html` — página principal com cards de módulos e painel interativo.
- `styles.css` — estilo visual para o layout de curso/streaming.
- `script.js` — navegação dos módulos e seleção de conteúdo.
- `site-structure.md` — plano do site e conteúdos iniciais.
- `pages/` — exemplos de páginas de conteúdo para cada módulo.
- `pages/eletronica.html` — landing page do módulo Eletrônica.
- `pages/informatica.html` — landing page do módulo Informática.
- `pages/arcade.html` — landing page do módulo Arcade.

## Como usar
1. Abra `index.html` em um navegador.
2. Clique em um dos cartões para abrir o módulo.
3. Selecione um conteúdo da lista para ver o preview.
4. Use os links de exemplo para acessar páginas de conteúdo individuais.
